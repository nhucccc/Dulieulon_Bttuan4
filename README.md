# Heart Disease Analysis using Apriori & K-Means

## Mô tả
Bài tập áp dụng:
- Luật kết hợp Apriori
- Phân cụm K-Means
trên bộ dữ liệu bệnh tim.

## Quy trình
1. Đọc dữ liệu CSV
2. Tiền xử lý dữ liệu
3. Áp dụng Apriori tìm luật kết hợp
4. Biến đổi dữ liệu one-hot
5. Phân cụm K-Means
6. Đánh giá bằng Silhouette Score
7. Trực quan hóa bằng PCA

## Công nghệ sử dụng
- Python
- Pandas, Scikit-learn
- Mlxtend
- Matplotlib

## Cách chạy
Mở file `.ipynb` và Run All.
